⚖ Gestão Advocacia

Configure uma vez e acesse de qualquer computador. O banco de dados fica no seu Firebase Firestore (gratuito).

1. Crie o projeto Firebase
Acesse console.firebase.google.com → "Adicionar projeto" → dê um nome → "Criar projeto".
2. Ative o Firestore
Menu lateral → Build → Firestore Database → "Create database" → Production mode → região southamerica-east1 → "Enable".
3. Regras de segurança
Na aba "Rules" substitua tudo por:
rules_version='2'; service cloud.firestore { match /databases/{d}/documents { match /{col=**} { allow read, write: if true; } } }
→ "Publish".
4. Obtenha as credenciais
⚙ Configurações do projeto → aba "Geral" → "Seus apps" → ícone </> → "Registrar app" → copie o bloco const firebaseConfig = { … } e cole abaixo.
Cole o firebaseConfig aqui:
